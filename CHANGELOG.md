## [Unreleased]

### Added
- Initial release
- Real-time I/O statistics visualization
- Multiple chart views (IOPS, Throughput, Utilization, Wait Times)
- Device switching with keyboard navigation
- Color-coded charts using curses
- Configurable refresh interval
- Cross-platform support (Linux, macOS)

### Changed
- Optimized for performance and maintainability
- Improved error handling and portability

### Added

- Comprehensive documentation
